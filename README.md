# project2_dice
+ https://slows14tem.github.io/project2.dice/

## 주사위 게임
+ 내가 선택한 숫자와 주사위 숫자가 일치하는지 확인
+ 자바스크립트 사용
+ 돔 display - none/block;
+ 난수 생성(1~6)으로 주사위 호출

## 개요
자바 스크립트를 사용하여 간단한 주사위 숫자 게임을 구현했다. 1~6까지의 렌덤 값을 생성하고 내가 선택한 값과 맞는지를 판단하는 게임이다.
